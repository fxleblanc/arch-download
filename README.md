# Download the latest arch linux iso
Python script to download and mount the latest arch iso.

# Usage
1. Clone the project
2. Launch the script as root
3. Profit!

# Support
This project was tested using the following system settings:
OS: 4.1.6-1-ARCH
Python: 3.4.3
